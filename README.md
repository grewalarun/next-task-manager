# 📋 Task Management App

A modern **Task Management Application** built with **Next.js** for the frontend, styled using **Tailwind CSS**, powered by a **REST API (Node.js + Express + MongoDB)** backend, and enhanced with **AI-powered text generation (Gemini)**.

---

## 🚀 Features

- ✅ Create, update, delete, and manage tasks
- 📂 Organize tasks with categories/status
- 🔍 Search and filter tasks
- ⚡ Fast and responsive UI using Tailwind CSS
- 🔐 RESTful API integration with Express & MongoDB
- 🤖 AI-powered task description generation using Gemini
- 📱 Fully responsive design

---

## 🏗️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### AI Integration
- Gemini API (Generative Text)
